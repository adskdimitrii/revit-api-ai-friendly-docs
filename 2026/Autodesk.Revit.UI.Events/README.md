# Autodesk.Revit.UI.Events

**1 file** | Assembly: `RevitAPIUI.dll`

## What you can do here

Use this namespace to **handle fabrication part browser events** in the Revit UI. It provides the enumeration needed to identify which type of operation is being performed in the fabrication part browser panel.

## Key outcome

- **Distinguish browser operations** — when handling a fabrication part browser event, use `FabricationPartBrowserOperation` to determine whether the user is selecting, filtering, or performing another action in the browser.
